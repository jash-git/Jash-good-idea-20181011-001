塑料瓶上綁2根筷子，放花盆裡，能解決家里花草容易乾枯的難題

資料來源: http://k.sina.com.cn/article_6519764898_1849bb7a200100adqm.html?cre=tianyi&mod=pcpager_focus&loc=1&r=9&doct=0&rfunc=31&tj=none&tr=9


礦泉水瓶大家都非常常見吧，我們在外面口渴時都會買一些這樣的水，但是我們把水喝光之後這個瓶子我們就扔掉了，其實這一個廢舊的塑料瓶我們還可以廢物利用，只需要簡單的改造一下它就可以變成一個小神器，趕緊跟著曉宇來看看吧。

首先我們把這個塑料瓶給它準備好，然後我們拿起這個壁紙刀，我們把底部給它割一下，我們可以把它割下來也可以把它不割下來，不割下來的話可以防止一些飛蟲等等進入，

然後我們再準備出一雙筷子，我們拿出膠帶給它固定一下，我們先把這個塑料瓶放到這個膠帶上面，然後拿出這個筷子，我們把它固定到這一個塑料瓶的兩端，要對稱一點兒，然後我們用膠帶給它粘一下把它固定住，

固定好後我們把這個蓋子給它擰下來，然後拿出這一個電烙鐵給它中間戳一個小孔，然後等它凝固一下，

我們就可以拿出一個手提袋的這樣一個繩子，我們把它穿過這一個小孔，我們只需要在裡面這樣一拽就可以了也不需要打結，然後我們把這一根繩子給它裁剪一下，裁剪完後我們就可以把它，引到我們這一個礦泉水瓶中了，這樣我們一個小神器就做好了，

在我們平時出差或者是沒有時間打理這個花草時，這個花草就會枯萎，但是這個花草又不易一次性澆太多的水，那麼該怎麼辦呢？這一個小神器就可以完全把它解決，我們只需要把它插到這一個花盆裡面就好了，然後把這個繩子放到這個花盆裡，我們只需要在這個塑料瓶中倒入清水，這個水就會順著這一個繩子慢慢的流下來了，這一個小神器就可以自動澆花了。
